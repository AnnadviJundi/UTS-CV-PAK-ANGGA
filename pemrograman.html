Materi Level Pemahaman Keterangan
HTML Element 4 Mampu membuat dan mengimplementasikan element HTML dalam berbagai kasus
HTML Tags 4 Paham penggunaan berbagai tags HTML dan implementasinya
HTML Atribut 4 Sudah mencoba membuat program dengan berbagai atribut HTML

Keterangan Level:

0 - Tidak tahu

1 - Tahu tapi belum mengerti

2 - Mengerti tapi belum dipraktekkan

3 - Sudah dipraktekkan tapi belum paham

4 - Paham sehingga bisa dibuat ulang dalam kasus yang berbeda
